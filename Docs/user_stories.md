# User Stories

## Normal User Stories (Planned for 2 Weeks)

1. As a user, I want to register an account so that I can create a profile and start using it.
2. As a user, I want to log in to the app.
3. As a user, I want to post a tweet .
4. As a user, I want to view a feed of freinds and others.
5. As a user, I want to follow and unfollow other users .
6. As a user, I want to like a tweet.
8. As a user, I want to comment  to a tweet .
9. As a user, I want to search for users to follow.
10. As a user, I want to view my own profile and tweets
11. As a user, I want to edit my profile (bio, photo, name) so that I can personalize my account.

## Premium User Stories :

1. As a premium user, I want all the previlages of a normal user
2. As a premium user, I want a blue tick

## Future Enhancements For Normal User:

1. As a user, I want to send direct messages to other users.
2. As a user, I want to bookmark tweets.
3. As a user, I want to see trending topics and hashtags.
4. As a user, I want to receive notifications for likes, replies, and new followers.

## Future Enhancements For Premium User:

1. As a premium user, I want my posts to reach followers more quicker than others
